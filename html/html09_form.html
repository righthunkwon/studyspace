폼 양식 만들기

        
        method : get(기본값, 내용 길이 제한 O, 주소표시줄에 입력 내용 드러남), post(내용 길이 제한 X, 입력 내용 안 드러남)

        
        text : 한 줄짜리 텍스트(로그인 id 입력, 글 제목 입력 등)
        password : 비밀번호 입력 필드
        email : 이메일 입력 필드
        radio : 주어진 여러 요소 중 하나 선택하는 버튼(태그 밖에 라벨 표시)
        checkbox : 주어진 여로 요소 중 두 개 이상 선택할 수 있는 버튼(태그 밖에 라벨 표시)
        checked : checkbox, radio에서 기본 선택 상태를 설정한다.
        button : 일반 버튼
        submit : 전송 버튼(폼 -> 서버)
        image : submit 버튼 대신에 이미지 삽입 가능
        
        name : form 컨트롤 요소의 값을 서버로 전송하기 위해 필요한 속성
        value : 기본 입력값 설정(아이디 저장)
        size : 입력 상자 크기(문자 수)
        maxlength : 최대 문자 수
        readonly : 읽기전용, 값이 들어가지 않음
        disabled : 비활성화
