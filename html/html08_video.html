비디오 재생

        src : 재생할 영상 파일 명
        controls : 재생 컨트롤러 표시
        width, height : 폭과 높이 지정(px, %)
        loop : 반복횟수
        muted : 음소거
        autoplay : 자동재생 (muted 시 자동재생 가능)
        preload : 파일 전송 완료 후 재생
        poster : 재생 전에 표시할 이미지(썸네일) - autoplay를 설정할 경우 영상이 바로 재생되므로 poster가 잘 구현되지 않음
